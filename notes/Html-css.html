Positioning

Four positioning methods:

  • Static: standard, according to flow of the page

  • Fixed: relative to browser window, stays put if page is scrolled. Other elements act as though these don’t exist.

  • Relative: relative to its normal (static) position. Often used as a container for absolute positions.

  • Absolute: relative to the first parent that is not static (html if there is none).

The position of the last three (other than static) can be controlled via the css attributes top, right, bottom, left.
