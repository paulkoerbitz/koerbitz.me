Differences from Propositional Logic

predicates

A predicate takes one or several entities as input and outputs either true or false. For example, in the statements ‘Plato is a philosopher’ and ‘Socrates is a philosopher’ ‘X is a philosopher’ is a predicate and X is instantiated to ‘Plato’ and ‘Socrates’.

quantification

Variables can be quantified over. For example ‘forall a: if a is a philosopher then a is a scholar’ expresses that this conditional statement holds for all a.
