Calc support in browsers

published on August 9, 2016

Fully supported from Firefox 16, Chrome 26, Safari 6.1, IE 10 (9 with a crash when used in background-position), iOS Safari 7.1.

A fallback for browsers not supporting this is to have two css rules for the attribute to set:

.class-to-style {
    height: 100%;
    height: calc(100% - 25px);
}
