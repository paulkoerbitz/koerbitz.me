How to bookmark file locations in emacs

published on February 17, 2016

Emacs has a bookmarking feature which allows to record positions in files and easily jump back to them afterwards. This can be extremely handy when navigating a large codebase.

  • C-x r m bookmarkname RET

    record a bookmark (mnemonic: record-mark)

  • C-x r b bookmarkname RET

    jump to a bookmark

  • C-x r l

    List all bookmarks (names and file locations), jump to them on enter.

More details at the Emacs manual.
